# TominaGarage - Landing Page

Social Work & Craftsmanship - Jeep Wrangler向けカスタムラゲッジボード専門店のランディングページ

## 技術スタック

- React 18
- Vite
- Tailwind CSS
- Lucide React Icons
- Google Imagen API（画像生成）

## セットアップ

```bash
# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev

# ビルド
npm run build
```

## 特徴

- スプラッシュスクリーン（ECGアニメーション付き）
- パララックス効果のヒーローセクション
- 斜めマーキースライダー
- スクロール表示アニメーション
- AI画像生成（Imagen API）
- レスポンシブデザイン

## API設定

画像生成機能を使用するには、`src/App.jsx`内の`apiKey`変数にGoogle Imagen APIキーを設定してください。
